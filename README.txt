# Minimalist Sentiment Analyzer ✨

A clean, interactive web application built with Python and Streamlit that analyzes the emotional tone of any given text.

## Overview

This project takes raw text input and utilizes Natural Language Processing (NLP) to calculate a polarity score, instantly determining whether the sentiment is Positive, Negative, or Neutral.

Instead of relying on a standard Jupyter Notebook, this project focuses on the end-user experience by wrapping the machine learning logic in a sleek, minimalist frontend.

## Tech Stack

- **Frontend:** Streamlit
- **NLP Engine:** TextBlob
- **Language:** Python

## How to Run Locally

1. Clone this repository.
2. Install the required dependencies:
   `pip install -r requirements.txt`
3. Launch the web app:
   `streamlit run app.py`
